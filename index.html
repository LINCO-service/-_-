✏️ 一、手動新增

適用情境

單筆新增，資料欄位逐一填寫
適合少量、需完整填寫的原料

注意事項

品名為唯一識別鍵，同一品名不可重複
舊品號（B欄）、新品號（C欄）為選填
採購狀態預設為「採購確認」

📥 二、全量匯入 可選擇採購狀態

處理流程

讀取 Excel
→
跳過第 1、2 列
(標題 + 說明行)
→
品名空白 / 含「略」/
開頭「勿」→ 跳過
→
比對現有資料庫
→
寫入

比對順序（優先度由高到低）

1 舊品號（B欄）
→
2 新品號（C欄）
→
3 品名（D欄）
三者比對均忽略大小寫與前後空白

找到現有記錄 方案 A 合併

既有欄位有值 → 保留原值，不覆蓋
既有欄位空白 → 補入匯入資料
id、createdAt 永遠保留
→ 計入：更新筆數

未找到 全新新增

建立新記錄，套用所選採購狀態
成本 > 0 時，同步寫入價格歷史
品名與其他品號記錄重複 → 略過並提示
→ 計入：新增筆數 / 略過筆數
⚠ 重要：Excel 格式必須符合規範，否則第一筆資料會被跳過
  • 第 1 列：欄位標題（日期、舊品號、新品號、品名…）
  • 第 2 列：固定放「勿刪 / 略 / 略」說明行（必須存在）
  • 資料從第 3 列起，N 列 = N 筆
  • C 欄（新品號）若為 #N/A 或 Excel 公式錯誤，自動視為空值

📤 三、部分匯入（補充或更新特定欄位）

找到現有記錄 強制覆蓋

匯入欄位有值 → 覆蓋現有值
匯入欄位空白 → 不動現有值
成本有變動 → 自動記錄價格歷史
採購狀態（procurementStatus）不會被更改
→ 計入：更新筆數

未找到 直接新增

建立新記錄（不需說明行格式）
無品名重複保護，重複品名也會新增
→ 計入：新增筆數
適用情境：原料已存在於資料庫，需補充或修正部分欄位（例如：補填採購成本、新增產地、更新食添證效期）。與全量匯入不同，有值的欄位會被覆蓋。

📊 四、匯出

匯出範圍

依目前篩選條件匯出（全部 / 待採購 / 已確認）
格式與匯入模板相同，可直接用於回填

匯出欄位

包含所有欄位（中英文標示、營養素、文件狀態等）
不含採購狀態欄位（procurementStatus）
建議做法：先匯出現有資料 → 在 Excel 中補充空白欄位 → 再用全量匯入（方案 A）回填，既有有值的欄位不會被清除。

⚖️ 五、全量匯入 vs. 部分匯入 快速比較

項目 全量匯入 部分匯入
找到現有記錄時 只補空白欄位，保留有值欄位 有值就覆蓋，空白才保留
品名重複保護 有　略過並跳出提示 無　直接新增，可能產生重複
採購狀態設定 可選擇　匯入前選待採購或已確認 不更改　維持原記錄的狀態
Excel 說明行（第 2 列） 必要　缺少則第一筆資料被跳過 非必要　直接從有資料的列開始
價格歷史記錄 成本 > 0 時，新增筆才記錄 成本有變動就記錄（新增或更新皆是）
適合情境 初次大量建檔、或補充空白欄位 更新既有記錄的特定欄位

📋 六、Excel 檔案格式規範

欄位對應（全量匯入）

欄位Excel 欄名說明必填
A日期建立日期，格式 YYYY-MM-DD 或 YYYY/MM/DD選填
B舊品號系統比對鍵（優先度最高），相同視為同一原料選填
C新品號系統比對鍵（優先度次之），#N/A 自動忽略選填
D品名系統比對鍵（最終）；空白、「略」、開頭「勿」的列會被跳過必填
E分類原料分類，用於過敏原自動偵測選填
M熱量（大卡）數值欄位，空白視為 0選填
N–W蛋白質 ~ 咖啡因各營養素（g 或 mg），空白視為 0選填
X–AB產規書 ~ 第三方檢驗報告文件狀態，有值視為已備齊選填
AC食添證效期 日期格式 YYYY-MM-DD 或 YYYY/MM/DD，自動截取前 10 碼
⚠ 建議將此欄儲存格格式設為「文字」，避免 Excel 自動轉換日期序號導致全量匯入與部分匯入結果不一致 		選填
AM廠商全稱供應商名稱選填
AW成本（kg-未稅）採購單價，匯入時同步寫入價格歷史（新增筆）選填
⚠ 全量匯入 Excel 固定格式
  • 第 1 列：欄位標題列（日期、舊品號、新品號、品名…），系統自動識別欄位名稱
  • 第 2 列：說明行，A欄固定填「勿刪」，其他欄填「略」（缺少此列，第一筆資料會被跳過）
  • 第 3 列起：實際原料資料，每列一筆
  • 部分匯入不需說明行，從有資料的列直接讀取
